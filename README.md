# Project Supernova

Project Supernova is a modular Next.js PWA for personal finance tracking. It
helps users manage budgets, track spending, and visualize financial trends with
a simple, intuitive interface.

## Features

- **User Authentication**: Secure sign-up and login with Supabase Auth
- **Budget Management**: Create, edit, and delete budgets by category
- **Spending Tracking**: Log and categorize transactions manually
- **Dashboard**: Overview of budgets, recent transactions, and spending trends
  with charts
- **Responsive Design**: Fully functional on desktop and mobile (React +
  Tailwind CSS)
- **Type Safety**: Built with TypeScript for reliability

## Why Project Supernova?

- Simplifies personal finance for beginners and experienced users
- Encourages healthy financial habits with actionable insights
- Fast, accessible, and motivational user experience
- Strong privacy and data security

## Tech Stack

- **Next.js** (App Router, PWA)
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Redux Toolkit**
- **Supabase Auth**
- **Drizzle ORM** (SQL migrations)
